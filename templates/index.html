Pitch-O-Rama!

Modeling the Outcome of a Pitch With Machine Learning


    The Wind Up

  • Datasets cleaned using Pandas in Jupyter Notebook
  • All data stored and connected using SQLite, Postgres, and AWS
  • Model created using Pandas and Scikit-Learn in Jupyter Notebook
  • Dashboard created using HTML/CSS, Flask, and minor in-line Javascript for functionality

    • The Pitch

  • Gradient Boosted Classifier model based off MLB Statcast data from 2019-2020
  • Only focused on top 9 pitchers in MLB due to computational barriers
  • ~71% accuracy in determining whether or not a pitch will be a 'strike'
